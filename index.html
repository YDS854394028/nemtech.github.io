NEM Developer Center

Start integrating NEM into your application.

NEM is a blockchain platform built from the ground up to be simple for developers to use. NEM gives you direct access to a specialized set of tested and secure on-blockchain features using your favourite programming language.

This documentation refers to NEM latest version, Catapult, which provides higher performance and new functionalities to NEM.
