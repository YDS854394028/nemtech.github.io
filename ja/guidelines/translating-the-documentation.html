    ドキュメントの翻訳¶

    この NEM 開発センターはドキュメント作成用 Python のフレームワークである Sphinx によって作成されています。その拡張である sphinx-intl は異なるドキュメントサイトをホストすることなく、ドキュメントを多言語に翻訳することができます。

    翻訳の追加/更新¶

    1. リポジトリをクローンして、Python 2.7 または 3.4+ と pip がインストールされていることを確認してください。
    $> git clone https://github.com/nemtech/nem2-docs.git
    1. プロジェクトの必須項目をインストール
    $> pip install -r requirements.txt
    1. .poファイルにドキュメントのメッセージを抽出
    $>  make gettext

    各ドキュメントファイルの .po ファイルは build/gettext に生成されます。

    1. 次のコマンドで source/locale へ翻訳ファイルをコピーしてください。これは英語のドキュメントが変更された時に、 .po ファイルを更新するためにも用います。
    $>  sphinx-intl update -p build/gettext -l <lang>

    <lang> をあなたの 言語コード に置き換えてください。

    1. source/locale/<lang>/LC_MESSAGES/ にある .po ファイルを翻訳してください。各 msgid は 翻訳された msgstr を提供します。
    2. 翻訳されたドキュメントの生成と変更の確認
    $> make -e SPHINXOPTS="-D language='<lang>'" livehtml

    Submit your changes into the nem2-docs repository.
